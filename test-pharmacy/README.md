# test-pharmacy

A CM (C Managed) project with C#-like syntax.

## Getting Started

### Build
```bash
cm build src/main.cm
```

### Run
```bash
cm run src/main.cm
```

## Project Structure

```
src/
  main.cm              # Entry point
  models/            # Data models
  services/          # Business logic
tests/               # Test files
public_html/         # Web assets
```
